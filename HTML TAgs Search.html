b tag

The b tag in HTML is udsed to draw users attension to element's content , which are not otherwise granted special importance. This was formarly known as Boldface element, and the most browsers still draw the text in boldface.

i tag

The i tag in HTML represents a range of text that is set off from the normal text for some reason, such as idiomatic text, technical terms, taxonomical designations, among others.

u tag

The u tag in the HTML represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline, but may be altered using CSS.

select tag

The select tag in HTML represents a control that provides a menu of options.

button tag

The select tag in HTML is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog.
